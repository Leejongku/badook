object-fit: cover 테스트 예시

아래 이미지는 object-fit: cover와 비율 박스를 적용하여
창 크기를 줄여도 여백 없이 꽉 차고, 비율이 유지됩니다.
(이미지의 일부가 잘릴 수 있습니다)

main_2.html 내 대표 이미지 예시

브라우저 창 크기를 줄여도 이미지가 깨지지 않고, 여백 없이 꽉 차게 보입니다.

.ratio-box {
  position: relative;
  width: 100%;
  padding-top: 56.25%; /* 16:9 비율 */
  overflow: hidden;
}
.ratio-box img {
  position: absolute;
  top: 0; left: 0; width: 100%; height: 100%;
  object-fit: cover;
}
